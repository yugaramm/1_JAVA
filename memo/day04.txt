▷switch문
	switch(변수명){
		case 1:
			실행할 문장;
			break;//if문을 제외한 가장 가까운 중괄호 영역 탈출
		case 2:
			실행할 문장;
			break;
		case 3:
			실행할 문장;
			break;
		case 4:
			실행할 문장;
			break;
		default:
			실행할 문장;
	}


삼항 연산자, if문, switch문의 비교
	삼항 : 조건식이 1개 있을 경우, 그리고 조건에 따라 값이 달라지게 하고 싶은 경우 사용
	if문 :	- 조건식에 비교연산자를 사용하거나 여러 개의 조건식을 논리연산자로 연결할 때 사용
		- 조건식을 여러 개 사용해야 할 때 사용
	switch문 : 하나의 변수에 여러 개의 값이 담길 수 있고, 각 값이 같은 지를 비교할 때 사용


대입 연산자(복합 대입 연산자, 누적 연산자)
	+=, -=, /=, *=, %=

	int money = 10000;
	money = money + 1000;
	money += 1000;

	int data = 10;
	data = data + 1;
	data += 1;
	data++; // 증감연산자

증감 연산자
	++, --

	전위형 : 코드 실행과 동시에 적용
		++data;
	후위형: 코드가 실행된 이 후 부터 적용
		data++;
