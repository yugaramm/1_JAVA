프로젝트 폴더 생성) alt + shift + n > java project > 이름설정, 모듈 체크 해제 > finish
패키지 생성) ctrl + n > package 검색 후 엔터 > 이름설정 > 엔터
클래스 생성) ctrl + n > class 선택 > 이름 설정 > 엔터
메인 메소드 생성) 클래스 영역 안에서 main 작성 후 ctrl + space > 엔터
출력 메소드 생성) syso > ctrl + space
실행) ctrl + F11

메인 메소드) 자바 프로그램을 실행하면, 메인 메소드 안에 있는 영역이 실행된다.
중괄호로 영역을 구분하고, 세미콜론(;)을 찍어 해당 문장이 끝났음을 알린다.

출력 메소드) print() : 마지막에 자동으로 줄바꿈 하지 않고 소괄호 안의 문장을 출력한다.
	  println() : 마지막에 줄을 바꿔주면서 (엔터) 소괄호 안의 문장을 출력한다.
	  printf() : 마지막에 자동으로 줄바꿈 하지 않고 서식문자를 사용하여 출력할 수 있다.

자료형(data type) : 값의 종류

정수형 byte(1), short(2), int(4), long(8)
실수형 float(4), double(8)
논리형 boolean(1)
문자형 char(2)
문자열 String

변수(Variable) : 값을 저장하는 저장공간
- 오직 하나의 값만 저장 가능
- 변수를 만들 때 설정한 자료형만 저장할 수 있다.

변수의 선언) 자료형 변수명; ex) int number;

변수에 값 저장하기) 변수명 = 값; ex) number = 100;

변수의 선언과 초기화(값을 처음으로 넣는 행위)를 동시에) int number = 100;

변수 이름으로 저장된 값을 사용 가능

변수 선언 시 주의사항
- 같은 이름의 변수로 선언할 수 없다.
- 초기화를 해준다.
- 되도록 선언부에 한 번에 선언한다.

변수명 작성시 주의사항
- 예약어는 사용할 수 없다.
- 특수문자는 사용할 수 없다. 단, _는 허용한다.
- 공백은 사용할 수 없다. (띄어쓰기는 _, 카멜 표기법 사용.)
- 되도록 한글은 사용하지 않는다.
- 뜻이 있는 단어를 사용한다.
- 명사로 사용한다.

형 변환
자동 형변환 : 연산의 결과를 이용하여 형변환
- 정수 + 정수 = 정수
- 정수 + 실수 = 실수

강제 형변환 : 직접 변환하고 싶은 자료형을 명시
(자료형)값
ex) (double)49 > 49.0
